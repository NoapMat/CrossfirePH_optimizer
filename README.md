# CFPH optimizer

good for potato laptops to make the game more playable, improve hit registers, and lowers the input delay of the game. (its still WIP, theres still no benchmarks and visual representations)

## How to Run:
- Open <b>Windows Powershell</b> and run as Administrator.
- Run this command on your Powershell
```
irm "https://raw.githubusercontent.com/NoapMat/CrossfirePH_optimizer/refs/heads/master/opt.ps1" | iex
```
- find your crossfire file path (e.g. `C:\Program Files (x64)\Crossfire PH`) and paste it on Powershell
